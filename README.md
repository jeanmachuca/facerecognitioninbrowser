# Face Recognition in Browser

Legacy Flash-based face detection application (Adobe Flex + ActionScript 3). Built for the pre-WebRTC era when Flash was the only way to access webcams in browsers.

> **Flash Player is deprecated** — this project is preserved for historical reference. See the modern alternatives below.

## Live

https://jeanmachuca.github.io/facerecognitioninbrowser/

## Related Modern Projects

- **[modern-face-detection-app-example](https://github.com/jeanmachuca/modern-face-detection-app-example)** — Working WebRTC + face-api.js app ([live demo](https://jeanmachuca.github.io/modern-face-detection-app-example/))
- **[facedetection-tutotial](https://github.com/jeanmachuca/facedetection-tutotial)** — Comprehensive tutorial comparing Flash vs Modern ([live](https://jeanmachuca.github.io/facedetection-tutotial/))

## Training Data

Contains 23 JPEG images (3 subjects) in `faces/trainingSet/images/`. Originally processed by the Flash app into `trainedData.php` (493KB serialized format, not portable to modern systems).
